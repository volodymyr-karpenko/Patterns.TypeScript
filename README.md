# Patterns.TypeScript

The project demonstrates 23 GoF software design patterns via TypeScript programming language using simplistic examples.

### Important

All of the Patterns are located in `src/patterns` directory.
The `UseCase` class and its base class `PatternUseCase` are related to none of the patterns and were created to deliver the output, produced by a pattern.

### View Output

Execute `npm i` (only once to install project dependencies).
Execute start script passing a pattern name as an argument enclosed in single quotes after a double hyphen (e.g. `npm start -- 'abstract factory'`).
